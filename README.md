# Moksh Vaghasia - Old Portfolio (v1)

The original HTML/CSS portfolio website for Moksh Vaghasia, built with Bootstrap 5 and vanilla CSS. This static site served as an early personal web presence before being replaced by the React-based portfolio at moksh555.github.io.

## Key Features

- **Hero Section** - Circular profile photo with name and tagline, styled with a beige-accented Bootstrap hero layout
- **About Me Card** - Bio paragraph highlighting software engineering background and goals, with a photo
- **Three-column Feature Links** - Quick-access cards for About Me, Projects (linking to GitHub), and Travelling sections
- **Travel Gallery** - Dedicated travelling.html page with a photo gallery of travel photographs
- **Contact Section** - Mail Me button that opens the user's default email client pre-addressed to the author
- **Multi-page Navigation** - Separate aboutMe.html and travelling.html pages linked from the landing page

## Tech Stack

| Layer | Technologies |
|---|---|
| Markup | HTML5 |
| Styling | CSS3, Bootstrap 5.3 |
| Fonts | Google Fonts (Bebas Neue, Libre Baskerville, Montserrat) |
| Icons | Bootstrap SVG icons |
| Hosting | GitHub Pages (via CNAME) |

## Setup and Installation

This is a fully static site with no build step required.

1. Clone the repository:
   ```bash
   git clone https://github.com/moksh555/portofolio_old.git
   cd portofolio_old
   ```
2. Open index.html directly in your browser, or serve it with any static file server:
   ```bash
   npx serve .
   ```

## Usage

- `index.html` - Landing page with hero section, about card, feature links, and contact button
- `aboutMe.html` - Expanded about me page
- `travelling.html` - Photo gallery of travel images

This repository is archived as a historical version. The current portfolio is available at https://moksh555.github.io
